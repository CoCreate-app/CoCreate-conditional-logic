CoCreate-conditional-logic

Show or Hide html elements triggered by a click or change action.

Install

npm install @cocreate/conditional-logic

Or you can use cdn link:

https://cdn.cocreate.app/latest/cocreate.min.js

Usage

This is conditional-logic reference

                                
    <h5 style='color:blue'>Toggle</h5>
    <div data-hide='#testdiv,#div2'  data-show='.testdiv,.div2'>
           <div  class="test">click me</div><br/>
    </div>
    <div id='testdiv' class="hidden testdiv"> Only SHOW </div>
    <div id="div2" class="hidden div2"> Only SHOW2</div>

This is conditional-logic reference

Attributes

  • data-show string optional

    selector of elements to display

  • data-hide string optional

    selector of elements to hide
